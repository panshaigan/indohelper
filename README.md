Indohelper
===============================
- Extract/clone to the ProjectInfinity directory
- run `update.bat eet` to download/update mods listed in eet.csv file
- ...or create your own csv file and run `update.bat your_csv_file`
- github, weaselmods, morpheus-mart, baldurs-gate.de supported for now
- at the first launch, power shell will install 7z plugin for extractions
